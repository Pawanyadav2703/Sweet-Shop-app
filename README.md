# Sweet-Shop-app
The Sweet Shop Management System is a full-stack web application developed to efficiently manage sweet products, inventory,user operations through a secure and interactive platform. 
A full-stack web application to manage sweets, inventory, and user operations.
The system supports secure user authentication, viewing and purchasing sweets, and admin-controlled inventory management.
It uses a RESTful backend API, persistent database, and a modern single-page frontend for a smooth user experience.

Note: Admin users can add, update, restock, and delete sweets, while users can browse, search, and purchase items.
